Etiquetas

La etiqueta p es para hacer un párrafo.

Con las etiquetas h (hasta 4) son para poner titulos.
Con la etiqueta br haces un salto sencillo, por ejemplo dentro de la etiqueta p para no tener que hacer otro párrafo.
La etiqueta b...esta etiqueta hace el texto en negrita.
Texto con strong...esta etiqueta es para definir una palabra o conjunto de ellas como importantes.
Texto con u...esta etiqueta subraya el texto.
Texto con ins... permite definir que un texto ha sido actualizado o insertado en el documento web.
La representación visual en el navegador suele ser la de texto subrayado.
Texto con i...muestra el texto marcado con un estilo en cursiva o italica.
Texto con em... es el apropiado para marcar con énfasis las partes importantes de un text, puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis.
Text con con strike... esta etiqueta tacha el texto.
Texto con del... el elemento del (deleted-borrado) marca las partes
de un texto o documento que han sido suprimidas o sustituidas.
Texto con small... hace el texto más pequeño.
Texto con big... hace el texto más grande.
Palabra1Texto con sub...hace el texto con subíndice.
Palabra2Texto con sup...y esta hace el texto con superíndice. 
Texto con pre...
usada para pegar texto manteniendo el pre formato propio del texto.
Texto con abbr...esta representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura.
Si está presente, el atributo title debe contener la descripción completa y nada más.
La etiqueta mark... resalta el texto.
Con la etiqueta a: "Titulo con enlace"con esta etiqueta puedes poner enlaces, cambiar el titulo que salga, o que salga en otra pestaña.

La etiqueta que permite mostrar el código preformateado (como se ve en el editor) es la etiqueta em.
